🤖 AI Life Compass – Your AI Guide for a Better Future

📌 Project Overview

AI Life Compass is an AI-powered web platform designed to help citizens discover opportunities and essential services through personalized recommendations. By analyzing user information such as age, education, occupation, income, location, interests, and skills, the platform provides intelligent suggestions for scholarships, jobs, government welfare schemes, healthcare benefits, skill development, and career guidance.

The project aims to bridge the gap between people and the opportunities available to them, making important information accessible through a simple, user-friendly interface.

🎯 Objectives

Provide personalized AI recommendations.

Help users find scholarships, jobs, and government schemes.

Support students, job seekers, farmers, senior citizens, and low-income families.

Improve awareness of welfare programs and career opportunities.

⚙️ Project Execution

1. The user visits the AI Life Compass website.


2. The user navigates through different sections using the navigation menu.


3. On the AI Recommendation page, the user enters personal details such as education, occupation, income, interests, and skills.


4. The JavaScript-based AI logic analyzes the entered information.


5. Based on predefined conditions, the system generates personalized recommendations such as:



Scholarships

Jobs

Government welfare schemes

Healthcare benefits

Skill development courses

6. The platform displays:



AI Recommendation

Reason for Recommendation

Confidence Score (Demo)

Suggested Next Steps

7. Previous recommendations are saved using Local Storage for future reference.
